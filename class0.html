Anatomy of a website

Your Content
+ HTML: Structure
+ CSS: Presentation
+ JS: Behavior & Interaction
= Your Website

A website is a way to present your content to the world, using HTML and CSS to present that content & make it look good.

Anatomy of an HTML page


  <!DOCTYPE html>
  <html>
    <head>
      <title>Title of the page</title>
      <link rel="stylesheet" type="text/css"
                             href="style.css"/>
    </head>
    <body>
      The page content here.
    </body>
  </html>

Anatomy of an HTML element

Anatomy of a CSS rule

Elements

HTML elements on on your page

HTML adds elements


  <div>Lorem ipsum <span>Dolor.</span></div>

CSS selects elements by name


  div {
    color: #000000;
    font-size: 14px;
  }
  span {
    color: #888888;
    text-decoration: underline;
  }

IDs

Items that appear only once on your page

HTML adds ids using the id attribute


  <div id="sidebar">
    My sidebar goes here.
  </div>

CSS selects ids using #


  #sidebar {
    background-color: #ff0000;
    width: 30%;
  }

Classes

Items that can appear many times on your page

HTML adds classes using the class attribute


  <div class="quote">
    Quote goes here.
  </div>

CSS selects ids using .


  .quote {
    color: #888888;
    font-size: 18px;
    font-weight: bold;
  }
